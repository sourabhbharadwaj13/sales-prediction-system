# sales-prediction-system
🔹 Overview
This project is a Machine Learning web app that predicts future sales using historical data.
It provides an interactive dashboard to analyze trends and make data-driven decisions.
🚀 Features
Sales Prediction using Machine Learning
Interactive Dashboard
Data Cleaning & Visualization
Simple and User-friendly Interface
🛠️ Tech Stack
Python
Streamlit
Pandas
Scikit-learn
▶️ Run the Project

pip install -r requirements.txt
streamlit run app.py
